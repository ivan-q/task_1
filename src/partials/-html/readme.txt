Проект был перенесён на Pug, поэтому шаблоны в формате html были сложены в эту папку, чтобы не мешали. Если потребуется их вернуть в проект, надо переместить их в ../partials и в файле tmpl.pug заменить расширения файлов в include с .pug на .html. Либо положить в папку page-index файл tmpl.html, который так же находится в этой папке (html)
